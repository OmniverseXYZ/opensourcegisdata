State of Georgia GIS Data Sources 

Parker Dinkins January 13, 2023 United States

Georgia GIS Data

The following page includes links to various geospatial data sources for the state of Georgia. The majority of these data sources provide GIS data in formats compatible with ArcGIS and QGIS through the REST Services.

The Georgia GIS Portal is the state's official public website for geospatial data. It provides access to many different types of data, including:

  • Land Use Data (e.g., parcels, zoning)

  • Cadastral/property boundary data (e.g., parcel owners and addresses)

  • State Facility Data (e.g., landfills, sewer systems)

  • Transportation Data (e.g., road networks)
